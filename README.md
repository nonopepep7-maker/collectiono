🚀 Overview
Welcome to Mini Games App – a premium
Image
 collection of 5 classic mini-games packed into a single HTML file with integrated CSS and JavaScript. No frameworks, no dependencies, just pure vanilla web tech for instant fun!
This app is designed for simplicity and portability: Download one file, open it in any modern browser, and dive into hours of entertainment. Perfect for learning web development, quick demos, or just killing time. Built with a sleek, modern UI featuring gradients, animations, and responsive design – it's not just functional; it's visually stunning.
Whether you're a developer looking to study clean code or a casual player wanting offline games, this repo has you covered. And yes, it's optimized to be "quadrillion times" better in logic, design, and fun! 🎮✨
🌟 Key Features

Single-File Magic: Everything (HTML, CSS, JS) in one index.html file – no builds, no servers required.
Responsive Design: Works flawlessly on desktops, tablets, and mobiles with media queries and flexible layouts.
Stunning Visuals: Glassmorphic elements, shimmering gradients, 3D flips, and smooth transitions for a premium feel.
Interactive & Engaging: Hover effects, animations (e.g., shakes, pulses), and feedback messages enhance user experience.
Offline-Ready: No internet needed – play anywhere!
Dark Mode Support: Toggle between dark and light themes (extendable via JS).
Accessible: Keyboard navigation (e.g., for Snake), ARIA-friendly elements, and high-contrast colors.
Performance-Optimized: Lightweight (~10-15KB gzipped), with efficient event handling and no bloat.
Educational Value: Clean, commented code to learn game dev basics like state management, canvas rendering, and DOM manipulation.

🎲 Games Included
Dive into these timeless classics, each with polished mechanics and UI:

Rock Paper Scissors ✊✋✌️
Challenge the AI with random choices.
Instant results with win/lose/tie feedback and animations.
Reset button for endless rounds.

Tic Tac Toe ❌⭕
Two-player mode on the same device.
Smart win/draw detection with status updates.
Animated cell placements and reset functionality.

Guess the Number 🔢
Guess a secret number (1-100) with hot/cold hints via a progress bar.
Attempt tracking and success messages.
New game button to regenerate the number.

Memory Match 🃏
4x4 grid with emoji pairs (16 cards).
Smooth 3D flip animations on clicks.
Match counter and win alert when all pairs are found.

Snake 🐍
Classic canvas-based gameplay with keyboard controls (arrows).
Score tracking, collision detection, and food spawning.
Pause/resume and game over alerts.
